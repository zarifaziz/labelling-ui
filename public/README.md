# Public Assets

This directory contains static assets served by Next.js.

## Sample CSV Files

Example datasets for testing the labelling tool:

- **`sample.csv`** - Original sample evaluation data
- **`activity-sample.csv`** - Activity-based evaluation data
- **`misconception-sample.csv`** - Misconception detection data

## SVG Icons

UI icons and branding:
- `file.svg` - File icon
- `globe.svg` - Globe/web icon  
- `next.svg` - Next.js logo
- `vercel.svg` - Vercel logo
- `window.svg` - Window/app icon

## Usage

These files are publicly accessible at `/filename` in the deployed app.

Example: `https://your-domain.com/sample.csv`
